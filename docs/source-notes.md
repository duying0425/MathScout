## 对设计的启示

- 教材换版有时效性。系统应存储版次标签与地区采用记录，并带上有效期与证据链接。
- 地区使用情况应尽量来自省/市/区教育局的通知，而非无出处的"教材版本地区表"帖子。
- 公开教学资源可作为抽取的种子，但 canonical 记录应保留来源出处与复核状态。
